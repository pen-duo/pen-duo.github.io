CSS动画

动画原理

定义

  • 动画由许多静止的画面(帧)组成
  • 以一定速度连续播放时
  • 肉眼因视觉残象产生错觉
  • 误以为是活动的画面

transfrom(变形)

CSS**transform**属性允许你旋转，缩放，倾斜或平移给定元素。

  • 位移translate
  • 旋转rotate
  • 缩放scale
  • 倾斜skew

注意:一般transform的使用都伴随着transition(过渡)

inline元素不支持transform

transform之translate

常见写法

transform:translateX(百分比或像素长)

transform:translateY(百分比或像素长)

transform:translateZ(百分比或像素长)且父容器perspective

transform:translate3d(x,y,z)

经验:

学会看懂MDN语法示例

translate(-50%,-50%)可作为绝对定位元素的居中。

transform之scale

常见写法

transform:scaleX(倍数)

transform:scaleY(倍数)

transform:scale(宽的倍数,高的倍数)

transform之rotate

常见写法

  • rotateX(xxxdeg)
  • rotateY(xxxdeg)
  • rotateZ(xxxdeg)

一般用于360度旋转制作loading

用到时再搜索rotateMDN看文档

transform之skew

skew() 函数定义了一个元素在二维平面上的倾斜转换.

skew() 函数指定一个或两个参数，它们表示在每个方向上应用的倾斜量。

常见写法

shewX()

shewY()

shew()

transform多重效果

组合使用

transform:scale(0.5) translate(-100%,-100%)

transform:none;取消所有

transition(过渡)

作用:补充中间帧

语法

transition:属性名 时长 过渡方式 延迟时长

例如:transition:left 200ms linear xxx(ms/s)

可以用逗号分隔两个不同属性

transition:left 200ms, top 400ms

可以用all代表所有属性

transition:all 200ms

过渡的方式有:linear | ease | ease-in |ease-out等等

注意:

并不是所有属性都能过渡

display: none =>block没法过渡

一般改成visibility:hidden =>visible

过渡必须要有起始,一般一次动画,最多两次。

animate(动画)

多次动画的结合使用animate

声明关键帧

添加动画

1.@keyframes xxx {

from {

some code

}

to {

some code

}

2.@keyframes xxx {

0%{some code}

30%{some code}

60%{some code}

100%{some code}

}

animation

缩写语法

animation:时长|过渡方式|延迟|次数|方向|填充模式|是否暂停|动画迷名

时长:1s或则1000ms

过渡方式:ease | linear等

次数:3或4或则infinite

方向:reverse | alternate | alternate-reverse

填充方式:none | forwards | backwards | both

是否暂停: paused | running

以上所有属性都有对于的单独属性
