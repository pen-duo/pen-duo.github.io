JS语法

1.语句

JavaScript 程序的执行单位为行（line），也就是一行一行地执行。一般情况下，每一行就是一个语句。

比如

var a = 1;

表达式（expression），指一个为了得到返回值的计算式

比如

1 + 3

两者的区别
前者主要为了进行某种操作，一般情况下不需要返回值；后者则是为了得到返回值，一定会返回一个值。

注意

表达式不需要分号结尾。一旦在表达式后面添加分号，则 JavaScript 引擎就将表达式视为语句，这样会产生一些没有任何意义的语句.

2.大小写敏感

比如

var a; var A;

上面两个变量是完全不同的两个变量。

值得注意的还有

object和Object

function和Function

3.空格与回车

在大部分时候加空格和回车是没有任何意义的,也不会产生任何影响,但是!!!

值得注意的是

不能再return 后面加回车(否则会返回undefined).

function person(){ return 1; } console.log(person())//undefined

4.标识符

第一个字符可以是Unicode字母,中文,下划线,$符。

注意标识符开头不能是数字,标识符也不能是关键字或保留字。

{}在语句只有一句的时候可以省略,不建议这样做。

10.三元表达式

console.log(“哈哈哈哈”);
} else {

console.log(“嘻嘻嘻嘻”);

}

a > b ? console.log(“哈哈哈哈”) console.log(“嘻嘻嘻嘻”)。

11.或,与,非

II 或

&& 与

! 非

或,与 按短路逻辑执行

或 从左到右判断时,遇到true就直接返回true,否则返回false.

与 从左到右判断时,遇到false就直接返回false,否则返回true.

12.while

while(表达式){语句}

判断表达式的正价

当表达式为真,执行语句,执行完在判断表达式

当表达式为假,跳过循环,执行后面的语句。

13.for循环

for(变量初始化,中止条件,增量) {

循环体

}

先执行语句1

然后判断表达式2,

如果为真,执行循环体,然后执行语句3

如果为假,直接退出循环,执行后面的语句

break退出所有循环

continue退出本次循环
