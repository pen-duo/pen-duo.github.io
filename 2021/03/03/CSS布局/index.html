CSS布局

float布局

步骤：

  • 子元素上加  float:left, 和width

  • 父元素加上 .clearfix

  • 父元素加上class: class=”clearfix”

  • .clearfix的写法：

.clearfix::after { content:’’; display:block; clear:both; }

一些经验：

  • 最后一个子元素不设置width， 或者设置max-width
  • img 设置Max-width
  • 如果图片下面有多余的空白，就在图片上写上 vertical-align: top或者middle
  • 如果border干扰了布局，可以用outline: 1px solid red, outline不占用位置
  • 固定元素的块级元素居中的方法： 写上margin: 0 auto; 或者margin-left: auto; margin-right: auto;(比前者好，因为没有影响上下的Margin)
  • 平均布局的关键点：负Margin

flex布局

flex布局两个主题:容器(contaniner)和项目(item)。

flex盒子两条轴:主轴和侧轴。默认为主轴x轴,侧轴为y轴。

起步:

display:flex将一个盒子设置成flex盒子。

容器属性:

flex-direction设置主轴方向。

justify-content设置项目在主轴上的排列方式。

align-items设置项目在侧轴上的排列方式(单行)。

align-content设置项目在侧轴上的排列方式(多行)。

flex-wrap设置项目是否换行。

flex-flow：主轴方向 是否换行。简写

  • flex-direction属性值

  • row---------->

  • row-reverse(从右到左)<---------

  • column(从上到下)

  • column-reverse(从下到上)

  • justify-content属性值

  • flex-start

  • flex-end

  • center

  • space-around(平均分成若干份)

  • space-between(一左一右紧贴两边,其余的平均分配剩余空间)

  • align-items属性值

  • flex-start

  • flex-end

  • center

  • stretch(所有项目高度默认与最高的项目一致)

  • space-around(平均分成若干份)

  • space-between(一左一右紧贴两边,其余的平均分配剩余空间)

  • align-content属性值

  • flex-start

  • flex-end

  • center

  • stretch(所有项目高度默认与最高的项目一致)

  • space-around(平均分成若干份)

  • space-between(一左一右紧贴两边,其余的平均分配剩余空间)

  • flex-wrap属性值

  • wrap(换行)

  • wrap-reverse(换行且颠倒)

  • nowrap(不换行)

项目属性:

order：决定此项目的排列优先级(越小越靠前)。

align-self：改变单个项目的排列方式。

flex-grow：指定了flex容器中剩余空间的多少应该分配给项目（flex增长系数）。

flex-shrink：指定了 flex 元素的收缩规则，仅在默认宽度之和大于容器的时候才会发生收缩，其收缩的大小是依据 flex-shrink 的值。

flex-basis：指定了 flex 元素在主轴方向上的初始大小。

flex: fle-grow flex-shrink flex-basis的简写。

align-self属性值

flex-start

flex-end

center

grid布局:新的table布局

起步:display:grid将盒子设置为grid盒子。

grid-template-columns: 40px 50px auto 50px 40px;

grid-template-rows: 60px 400px 200px;

grid用法总结：

  1. 用grid-template-areas 设计大致的布局

  2. 示例：

    grid-template-areas: “big mid1” “big mid2” “sm1 mid2” “sm2 mid3” “sm3 mid3”

  3. 用grid-template-rows和grid-template-columns指定每一行每一列的高度和宽度

  4. 示例：

  5. grid-template-rows: 240px repeat(4, 120px);

  6. grid-template-columns: 250px 250px;

  7. 对于每一个小块，用grid-area和grid-template-areas 里布局的字符串对应

示例：

.demo > .image:first-child{ grid-area: big; border: 1px solid red; }
