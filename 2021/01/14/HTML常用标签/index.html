HTML常用标签

前言:今天我们来了解一下HTML的常用标签

HTML常用标签

1.a标签

属性

  • href
  • target
  • download(不常使用)
  • rel=noopener

href的取值

  • 网址
    • https://www.google.com
    • http://www.google.com
    • //google.com
  • 路径
    • /a/b/c或./a/b/c
    • index.html以及./index.html
  • 伪协议
    • javascript:代码;
    • mailto:邮箱地址
    • tel:手机号
  • id
  • href=#xxx

注意:以后调试不直接双击页面打开,而是使用http-server包打开。
因为:直接双击打开路径是从file(本地根目录)开始,而使用http-server是从开启hppt服务的根目录开始。
使用方法:http-server . -c-1

target的取值

  • 内置名字
    • _blank(新页面打开)
    • _self(原页面打开)
    • _top(如果页面中嵌套了框架,从页面最顶层打开)
    • _parent(如果页面嵌套了框架,从当前页面的上一级页面的打开)

2.iframe

内嵌窗口,现在很少使用了,只有一些老系统还在使用

3.table标签

表格
相关的标签

  • table
  • thead
  • tbody
  • tfoot
  • tr
  • th
  • td
    相关的样式
  • table-layout:auto/fixed(表格宽度是否根据字体数自适应)
  • border-collapse(合并两个表格的边框)
  • border-spacing(两个表格之间的间距)

4.img标签

作用:发送get请求,展示以上图片
属性

  • src
  • alt
  • height
  • width
    事件
  • onload
  • onerror
    响应式
    max-width:100%
    可替换元素:[][https://developer.mozilla.org/zh-CN/docs/Web/CSS/Replaced_element]

5.form表单

作用:发送get或post请求,然后刷新页面
属性

  • action
  • autocomplete
  • method
  • target
    事件
    onsubmit

6.input标签

作用:让用户输入内容
属性:
类型type:text/password/radio/checkbox/submit/number/tel/button/email/file/hidden/search
其他:disabled/checked/placeholder/value/name/maxlength/autofocus
事件

  • onchange
  • onfocus
  • onblur
    验证器(html5新增)
    required

7.其他输入标签

  • select+option
  • textarea
  • label
    注意事项:
    一般不监听input的click事件
    form里面的input要用name属性
    form里面要放一个submit的input才能触发submit事件

8.其他标签

标签
video
audio
canvas
svg

注意事项:
这些标签的兼容性一定得查看文档
